This is the address book. It is used as a way to enter and store data. When a user lands on this page there will be a form to add contact information such as (name, city, address, email, phone number, and hometown). When the user clicks the add button at the top of the webpage it will open a form and have 2 buttons at the bottom. After the user has clicked the add button they have the option to input and store information with the "Add Now" button (this button is tied to the addToBook function that will push the data to local storage), or they can cancel with the "cancel" button. If they cancel it will delete any information that was typed and nothing will be saved. This is becasue we have a clearForm function that will delete the filled but not submitted form form memory. If all of the fields are not filled out it will not submit the form by using an addToBook function that has and isNull statement that will check if all the fields have dont have an empty string value. Using the showAddressBook function it will display the address on the webpage if it is in the local storage. This webpage uses onClick event listeners to know when to use these functions that are associated with the proper button. This form also has a delete function that will delete the data from storage with the removeEntry function. THe jsonstructure function basically allows use to contruct variables and create objects.
